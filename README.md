## 📖 About The Project

Image to Grayscale Converter is a simple Computer Vision application developed using Python and OpenCV. The project demonstrates how colored images can be transformed into grayscale images through image processing techniques.

Grayscale conversion is one of the most fundamental operations in Computer Vision and serves as a preprocessing step for many advanced image analysis and machine learning applications.
